## Project Title
PhibyDay Coffee - Company Profile Website

## Description
PhibyDay Coffee adalah situs company profile fiktif sederhana yang memperkenalkan brand kopi dengan tampilan modern dan elegen khas anak muda.  
Website ini dibuat untuk memberikan informasi seputar latar belakang, menu unggulan, dan kontak PhibyDay Coffee secara interaktif tanpa memerlukan backend.  

Proyek ini dikembangkan sebagai bagian dari Capstone Project, dengan tujuan melatih pembuatan website statis yang dapat dideploy secara online.

---

## Technologies Used
1. HTML5 – untuk menyusun struktur halaman (bagian Home, About, Menu, Contact)
2. CSS – untuk mempercantik tampilan
3. GitHub – untuk manajemen kode dan version control
4. Netlify – untuk proses deployment website agar dapat diakses secara publik

Alasan pemilihan teknologi:
Teknologi ini dipilih karena ringan, mudah digunakan oleh pemula, dan sangat cocok untuk membuat website statis tanpa memerlukan backend atau database.

---

## Features
1. *Home Section* – menampilkan slogan utama “Enjoy your coffee with PhibyDay”  
2. *About Us Section* – menjelaskan sejarah singkat dan identitas brand  
3. *Menu Section* – menampilkan tiga varian minuman populer dengan gambar dan deskripsi singkat  
4. *Contact Section* – berisi alamat, nomor telepon, dan media sosial  
5. *Responsive Layout* – tampilan menyesuaikan layar ponsel maupun laptop  

Semua fitur dirancang agar pengguna dapat mengenal PhibyDay Coffee dengan cepat dan mudah.
